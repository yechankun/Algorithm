# 문제 링크

[14501번: 퇴사](https://www.acmicpc.net/problem/14501)

# 문제 풀이

1. DP로 문제를 푼다.
2.

# 풀이 코드

```python
import sys
input = sys.stdin.readline

def solution():
    N = int(input())
    TP = [list(map(int, input().split())) for _ in range(N)]

    dp = [0] * (N+1)

    for i in range(N):
        for j in range(i+TP[i][0], N + 1):
            if dp[j] < TP[i][1] + dp[i]:
                dp[j] = TP[i][1] + dp[i]
    return dp[-1]

print(solution())
```
